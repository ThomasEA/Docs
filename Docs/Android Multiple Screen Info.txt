Especificações para implementação de app com suporte a múltiplos dispositivos:

Logical Density		Friendly Name		Scale
(dpi)
=====================================================
120			LDPI			0.75x
160			MDPI			1x	--> "baseline"
240			HDPI			1.5x
320			XHDPI			2x
480			XXHDPI			3x
640			XXXHDPI			4x

*A medida média para um controle que necessita de interação através do touch é 50dp.

*Os qualificadores de layout podem ser:
	small
	normal
	large
	xlarge

	ou

	sw600dp
	w600dp
	h600dp
	sw700dp

*Também é possível criar Values contendo os valores específicos para multipos dispositivos:
	values-small
	values-normal
	values-large
	values-xlarge
	values-sw320dp
	values-sw360dp
	...
	...

		** Até a API 13 é utilizado a nomenclatura com sufixo small, normal, large e xlarge.
		   Após a API 13, são utilizadas nomenclaturas conforme o tamanho da tela: sw360dp, w600dp, h720dp


*Calculo de DP: px = dp * (dpi/160)
